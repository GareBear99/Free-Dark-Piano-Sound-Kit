## Loop Categories

### Classical
These are based on **public-domain classical material**, with extra emphasis on minor-key, gothic, funeral, requiem, and haunted-piano tones so they sit better beside the dark originals.

Included:
- Fur Elise motif
- Moonlight Sonata texture
- Ode to Joy
- Funeral March
- Toccata in D Minor motif
- Lacrimosa descend
- Prelude in E Minor texture
